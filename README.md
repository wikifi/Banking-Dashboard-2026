# Banking Dashboard – Power BI Project

## Overview

This project is an interactive **Banking Dashboard** built using **Microsoft Power BI**. The dashboard provides insights into banking operations, customer deposits, and loan analysis through dynamic visualizations and KPIs.

The report is designed to help users:

* Monitor banking performance metrics
* Analyze loan distribution and trends
* Track customer deposits
* View summarized banking insights in a single interface

---

# Dashboard Pages

## 1. Home

The Home page acts as the landing page of the dashboard and provides:

* High-level KPI cards
* Interactive slicers for filtering data
* Navigation buttons to move between report pages
* Overall banking performance overview

### Key Components

* KPI Cards
* Filters/Slicers
* Navigation Buttons
* Summary Metrics

---

This page focuses on loan-related insights and analytics.

### Features

* Loan performance KPIs

* Comparative analysis using charts

### Visuals Used

* Column Charts

* Total Loans Issued

# How to Use

1. Open the `.pbix` file in **Microsoft Power BI Desktop**.
2. Refresh the dataset if required.
3. Use slicers and filters to interact with the dashboard.
4. Navigate between pages using buttons.
5. Analyze KPIs and charts for banking insights.
